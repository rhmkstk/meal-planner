# Weakly Meal Planner

Nuxt project configured with Tailwind CSS and Nuxt UI.

## Requirements

- Node.js 22.12.0 or newer

## Setup

```bash
npm install
```

## Environment

For the Nuxt app, set these variables in local `.env` and Cloudflare:

```bash
NUXT_PUBLIC_SUPABASE_URL=
NUXT_PUBLIC_SUPABASE_KEY=
```

## Development

```bash
npm run dev
```
